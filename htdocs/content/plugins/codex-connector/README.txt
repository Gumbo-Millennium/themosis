== Description ==

Codex Connector is de Wordpres plug-in van de administratieve dienst Codex. Hiermee kan je alles wat je beheert in Codex, ook op een makkelijke manier op je Wordpress website integreren. Zo blijft het beheer aan de achterkant en regel je in Wordpress de weergave. *Delegeren kan je leren!*

== Installation ==
### Installeren en Configureren
Installeren kan via de Wordpres plug-in directory of als je deze plug-in handmatig hebt gedownload, volg dan de volgende stappen:

1. Upload `codex-connector.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

Vervolgens moet je de plug-in nog instellen. Doe dit als volgt:
1. Log in op jouw Codex en kijk in de module **Wordpress** voor de juiste gegevens.
2. Ga naar de pagina **Instellingen** > **Codex Connector** in de WordPress beheer omgeving
3. Vul daar de gegevens van stap 1 in en test de verbinding.

Vervolgens kan je in je templates en in je posts Codex-gerelateerde informatie weergeven.
### Codex Data in Templates
Je kan hier zogenamde *template tags* voor gebruiken. Kijk in de **WordPress** module in je Codex voor een lijst van beschikbare tags.
Je kan vervolgens in je eigen templates de volgende code verwerken: ``<?php do_action('codex_TAG'); ?>``

### Codex Data in Posts
Je kan hier zogenamde *shorttags* voor gebruiken. Kijk in de **WordPress** module in je Codex voor een lijst van beschikbare shorttags.

== Arbitrary section ==

Gebruik van deze plug-in en de WordPress module in COdex betekent dat de gebruikers akkoord gaan met de voorwaarden die gelden voor developers die werken met Codex.  Codex is op geen enkele manier aanpsrakelijk voor enig resultaat van het gebruiken van deze plug-in op externe websites. De gebruiker draagt zorg om discreet en voorzichtig om te gaan met enige gevoelige gegevens die door middel van deze plug-in en aanverwante producten uit de administratie van de organisatie te halen is. Bij het stoppen van de samenwerking tussen Codex en de organisatie dient de gebruiker deze plug-in te verwijderen van haar website.
